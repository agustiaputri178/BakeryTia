Langkah-langkah Upload Gambar

1
Persiapkan Gambar

Siapkan gambar produk dengan format JPG, PNG, atau GIF. Ukuran maksimal 2MB.

2
Pilih File

Di form tambah produk, klik area upload gambar atau tombol "Choose File".

3
Upload

Pilih gambar dari komputer Anda, lalu klik "Open". Gambar akan otomatis diupload.

4
Preview

Setelah upload, preview gambar akan muncul. Pastikan gambar sudah sesuai.

üí° Catatan Penting:

Untuk demo website statis ini, gambar akan disimpan di browser Anda menggunakan localStorage. Di website production sebenarnya, gambar perlu diupload ke server.

Solusi Jika Upload Tidak Berfungsi

Jika fitur upload tidak berfungsi (karena website statis), Anda bisa:

  1. Upload gambar ke hosting gambar gratis seperti Imgur
  2. Copy URL gambar tersebut
  3. Tempel URL di form admin (kolom gambar)
  4. Format: https://i.imgur.com/namafile.jpg
